# 7UP Website Clone

A front-end clone of the 7UP website built using **HTML** and **CSS**. This project was created as part of my web development learning journey to improve my understanding of website layouts, styling, and responsive design.

## 🚀 Overview

The goal of this project was to recreate the visual appearance and structure of the 7UP website while gaining hands-on experience with HTML and CSS. Through this project, I learned how professional websites are designed and how different sections work together to create a complete user experience.

## 🛠️ Built With

- HTML5
- CSS3

## ✨ Features

- Responsive webpage design
- Navigation bar
- Hero section
- Product showcase sections
- Promotional content sections
- Image integration
- Footer section
- Clean and organized code structure

## 📚 Learning Outcomes

Through this project, I learned:

- Semantic HTML elements
- CSS styling and layouts
- Flexbox positioning
- Responsive web design principles
- Website structure and content organization
- Image handling and optimization
- Writing clean and maintainable front-end code

## 🎯 Purpose

This project was developed for educational purposes to strengthen my front-end development skills and gain practical experience by recreating a real-world website design.
